IPL Overview Dashboard | Power BI
📌 Project Overview

The IPL Overview Dashboard is an interactive Power BI project that analyzes Indian Premier League (IPL) match and player statistics using historical IPL data. The dashboard provides insights into team performance, batting records, bowling records, toss impact, and overall tournament statistics through interactive visualizations.

This project demonstrates my skills in Power BI, Data Cleaning, Data Modeling, DAX, Data Visualization, and Dashboard Design.

📈 Business Insights
Mumbai Indians have recorded the highest number of match victories among the teams included in the dataset.
Winning the toss does not always guarantee winning the match.
Bowling statistics help identify players contributing through catches, run-outs, and dismissals.
Interactive slicers allow season-wise and team-wise performance analysis.
The dashboard enables quick comparison of overall IPL statistics.
