🌍 Global GDP Analytics Dashboard (2020–2025)

An interactive Power BI dashboard built to analyze and visualize global GDP trends from 2020 to 2025. This project provides deep insights into country-wise GDP performance, GDP growth trends, global economic share, and GDP per capita comparisons through visually engaging and dynamic reports.

🚀 Features
📊 Country-wise GDP Analysis
🌎 Global Share Visualization
📈 GDP Growth Trends (2020–2025)
💰 GDP Per Capita Comparison
🗺️ Interactive World Map Visualization
🔍 Dynamic Country Filters & Slicers
📌 KPI Cards for:
Total Countries
Total GDP
Average GDP
GDP Difference
Global Share Metrics
🛠️ Tools & Technologies Used
Power BI
Excel
DAX (Data Analysis Expressions)
Data Visualization
Data Cleaning & Transformation
