# Adidas Sales Analysis Dashboard (Power BI)

## 📊 Project Overview

The **Adidas Sales Analysis Dashboard** is an interactive business intelligence project built using **Microsoft Power BI**.
This dashboard provides insights into Adidas sales performance across different regions, products, retailers, and time periods.

The goal of this project is to transform raw sales data into meaningful insights that help understand sales trends, profitability, and product performance.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing products and regions
* Track sales trends over time
* Understand retailer contributions
* Visualize key business metrics for better decision-making

---

## 🛠 Tools & Technologies

* **Power BI Desktop**
* **DAX (Data Analysis Expressions)**
* **Power Query**
* **Data Visualization Techniques**

---

## 📌 Key Dashboard Features

* **Total Sales Overview**
* **Profit Analysis**
* **Units Sold Tracking**
* **Sales by Region**
* **Sales by Product Category**
* **Retailer Performance Analysis**
* **Monthly and Yearly Sales Trends**
* Interactive **filters and slicers** for dynamic analysis

---

## 📈 Key Insights

* Identifies the **top-performing regions and products**
* Helps track **monthly and yearly sales growth**
* Provides insights into **retailer contributions**
