## 📌 Project Overview

The **Bank Loan Analysis Dashboard** is a finance analytics project designed to extract meaningful insights from customer loan data using **MySQL** and **Microsoft Excel**.

The project focuses on analyzing loan performance, customer verification status, revolving balances, and home ownership patterns through interactive KPI visualizations.

---

## 🎯 Objectives

* Analyze loan amount statistics across employment years.
* Evaluate revolving balances based on loan grades and sub-grades.
* Compare total payments made by verified and non-verified customers.
* Study loan status by state and last credit pull date.
* Analyze home ownership trends and payment behavior.
