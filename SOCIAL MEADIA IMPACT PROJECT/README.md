# 📊 Social Media Impact Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing the impact of social media usage on **sleep, productivity, and mood** using an interactive dashboard built in Power BI.

The goal of this project is to uncover patterns in user behavior and provide meaningful insights into how excessive screen time affects daily life.

---

## 🎯 Objectives

* Analyze daily social media usage trends
* Understand the relationship between screen time and sleep
* Evaluate how social media affects productivity
* Identify behavioral patterns across different users

---

## 📂 Dataset

The dataset used in this project is a custom-built dataset that includes:

* Name
* Date
* App (Instagram, YouTube)
* Screen Time (hours)
* Sleep Hours
* Productivity Hours
* Mood
* Time Slot (Morning, Afternoon, Evening, Night)

---

## 📊 Dashboard Features

* 📈 Screen Time Trend Analysis
* 🥧 App Usage Distribution
* 📉 Screen Time vs Sleep Relationship
* 📊 Productivity Analysis
* 😊 Mood-Based Insights
* ⏰ Peak Usage Time Analysis
* 👤 User-wise Comparison

---

## 🔧 Tools & Technologies

* Power BI
* Microsoft Excel

---

## 📌 Key Insights

* Higher screen time is associated with reduced sleep duration
* Night-time usage is significantly higher than other time slots
* Increased usage often leads to lower productivity
* Certain users show higher levels of social media dependency

---

## 🚀 How to Use

1. Download the Power BI (.pbix) file from this repository
2. Open it using Power BI Desktop
3. Use slicers to filter by:

   * Name
   * Date
   * App
   * Mood
   * Time Slot
