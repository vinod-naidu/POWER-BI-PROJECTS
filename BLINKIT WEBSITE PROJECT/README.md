# 🛒 Blinkit Sales Analysis Dashboard

## 📊 Project Overview

The **Blinkit Sales Analysis Dashboard** is a data analytics project built to analyze sales performance, product categories, and outlet distribution for Blinkit – India's last-minute grocery delivery platform.

This dashboard provides insights into sales trends, item categories, outlet types, and customer preferences using interactive visualizations.

The project helps understand how different factors like **fat content, outlet size, location, and product type** affect overall sales performance.

---

## 🚀 Key Insights

* Total sales performance across different outlets
* Sales distribution based on **fat content**
* Sales analysis by **item type**
* Outlet performance by **location tier**
* Impact of **outlet size** on total sales
* Sales trends based on **outlet establishment year**

* Meat
